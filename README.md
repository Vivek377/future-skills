# Help Center Cards App

## Installation

1. **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/help-center-cards-app.git
    cd help-center-cards-app
    ```

2. **Move in to frontend folder**
    ```bash
    cd frontend
    ```

3. **Install the dependencies:**
    ```bash
    npm install
    ```

4. **Run the app**
   ```bash
   npm run dev
   ```
