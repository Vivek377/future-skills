# Help Center Cards App

## Installation

**Clone the repository:**
    ```
    git clone https://github.com/Vivek377/future-skills.git
    ```


**To run frontend of the app**

    ```b
    cd frontend
    ```

**Install the dependencies:**

    ```b
    npm install
    ```

**Run the app**

   ```
   npm run dev
   ```

**To run backend of the app**

    ```b
    cd frontend
    ```

**Install the dependencies:**
    ```b
    npm install
    ```

**Run the app**
   ```bash
   npm run dev
   ```
