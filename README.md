# Help Center Cards App

## Installation

**Clone the repository:**
    ```bash
    git clone https://github.com/Vivek377/future-skills.git
    ```


**To run frontend of the app**
    ```bash
    cd frontend
    ```

**Install the dependencies:**
    ```bash
    npm install
    ```

**Run the app**
   ```bash
   npm run dev
   ```

**To run backend of the app**
    ```bash
    cd frontend
    ```

**Install the dependencies:**
    ```bash
    npm install
    ```

**Run the app**
   ```bash
   npm run dev
   ```
